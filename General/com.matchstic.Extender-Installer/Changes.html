0.3.4

  • Resolved issue where if an application is re-signed when still within the 7-day window, it would not gain a new expiry date.

  • Resolved issue where re-signs would continually re-occur until the 7-day window is passed.

  • Added automatic queuing of re-signs if the device is locked.

  • Added various safety checks on installation.

  • Added UI indicator in-app of a re-sign’s progress.

    • +

    • &

    • @

    • -

    All other symbols should also working now.

  • Resolved issue causing Auto-Revoke Certificates to not function as expected.

  • Resolved issue causing Saurik's code to run twice for a given application when trying to re-sign it.
